# ASP.NET MVC Dinamik Haber Portalı

Bu proje, **ASP.NET MVC** mimarisi kullanılarak geliştirilmiş, dinamik içerik yönetimine sahip bir haber sitesidir.

## 🚀 Teknolojiler
* **Backend:** C#, ASP.NET MVC, Entity Framework Core
* **Veritabanı:** MSSQL
* **Frontend:** HTML, CSS, Bootstrap, JavaScript

## ⚙️ Özellikler
* **Yönetim Paneli (Admin):** Haber ekleme, düzenleme, silme ve kategori yönetimi.
* **Kullanıcı Sistemi:** Üye olma, giriş yapma ve çıkış işlemleri.
* **Yorum Sistemi:** Üyeler haberlere yorum yapabilir.
* **Kategoriler:** Haberler spor, teknoloji, ekonomi gibi kategorilere ayrılmıştır.
* **Dinamik Vitrin:** Ana sayfada son eklenen haberler ve manşetler otomatik listelenir.
* **Güvenlik:** Yetkilendirme (Authorize) ve CSRF koruması.
